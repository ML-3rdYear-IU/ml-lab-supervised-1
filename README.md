## Datasets

- **iris.csv**: Contains 150 samples, 4 features (SepalLengthCm, SepalWidthCm, PetalLengthCm, PetalWidthCm) and a target `Species`  
- Features = columns, Samples = rows, Target = class labels

---

## Homework Assignment

Students should implement:

1. **Logistic Regression**
   - Train/test split  
   - Evaluate accuracy, precision, recall  
   - Optional: plot 2D decision boundaries

2. **Support Vector Machine (SVM)**
   - Train/test split  
   - Evaluate performance  
   - Optional: visualize 2D decision boundaries

> Use the same dataset `datasets/iris.csv`. Templates are provided in the `homework/` folder.
